Linear Regression (Part 1) - Introduction

Posted on 2018-08-14 | Post modified: 2018-08-15 | In ML

介紹線性模型中的 Linear Regression。

Introduction

線性迴歸(Linear Regression)利用線性模型，來解決 regression 的問題，屬於監督式學習。

假定原始資料中共有 d 個 feature，即一筆資料 $ x = (x_1, x_2, …, x_d) $ ，則一個最基本的線性模型試圖用一個所有feature的線性組合來描述我們想要預測的目標，在 regression 問題當中，目標是一個連續性的值，例如氣溫、價錢等等。

$$ f(x) = w_1x_1 + w_2x_2 + … + w_dx_d + b，使得 f(x_i) \simeq y_i $$
用向量形式可寫為: $ f(x) = W^Tx + b $
其中每一個 $w$ 是每一項特徵的 weight ， $b$ 則是一個數(bias)，所要學習的就是 $w$ 和 $b$ 的值，線性模型較為基礎簡單，但好處是有非常好的可解釋性質，根據每個 feature 對應的 $w$ ，能很好的判斷哪個feature較重要。

如下圖所示，假設每筆資料包含2個 feature ($x_1, x_2$)，則這兩維度和目標y形成的三維空間中，$f(x) = b + w_1x_1 + w_2x_2$這個線性方程式所形成的線性模型即為一個平面，而我們要的目標是讓這個平面越符合y的分佈越好。

那要如何判斷分布的好不好?可以看所有資料點的y到這個平面的總距離和，這個距離和越小越好，也就是整體而言更接近我們線性模型表達的超平面，以圖示為例，目的就是找到一個平面能讓圖中的紅線總和有最小值。

Solution

如何找到一組最適合的參數(weight、bias)來有效的fit資料，主要有兩種方法:

  1. 用 gradient descent 來 minimize cost function，不斷迭代找到最好的組合。
  2. 直接用矩陣的運算找出最佳解(Nomral Equation)。

兩種方法都能有效找出解，normal equation的好處在於計算迅速、有效率，但只適用於線性迴歸，而 gradient descent 的概念則是在 Logistic Regression、神經網路等等都會使用到。
