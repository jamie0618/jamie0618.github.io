Keras Callbacks 紀錄

Posted on 2018-04-27 | Post modified: 2018-07-14 | In Keras

紀錄如何在 Keras model 中使用各種 callbacks

Checkpoint

官方文檔

  • filepath : 保存的路徑
  • monitor : 根據哪個值來決定，預設是’val_loss’，可自行改為’val_acc’等等
  • save_best_only : 預設為False，若設為True則會記錄最好的那個model
  • mode : 可以設定為 ‘max’ 或 ‘min’，用於當 save_best_only=True 時決定哪個是最好的判斷標準，如使用 ‘val_acc’ 時就選擇為 ‘max’
  • save_weights_only : 預設 False，會儲存整個模型架構和參數
  • period : 每隔幾個 epoch 檢查一次 checkpoint 動作
